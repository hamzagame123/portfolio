As an interaction designer, I obsess over the milliseconds. The slight bounce of a button, the easing curve of a modal, the feedback loop of a form submission.

I use tools like Figma, Protopie, and Webflow to build high-fidelity prototypes that feel real before a single line of production code is written. My goal is to reduce cognitive load while maximizing delight.
